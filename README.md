# Damascene Art - E-commerce Platform

## Project Overview
Damascene Art is a premium handcrafted brand selling Damascus-style wooden boxes inlaid with mother-of-pearl.  
This repository contains an academic junior project and a future commercial-ready platform.

## Goals
- **Academic:** deliver a complete junior project with solid requirements, architecture, and documentation.
- **Business:** evolve into a production e-commerce platform for the brand.
